# Golden Garden

A relaxing gardening game made with Python and Pygame.

>>> Features

- Plant and harvest flowers
- Earn BloomBits
- Buy seeds
- Save and load progress
- Animated flowers
- Inventory system

>>> Controls

- Left Click = Interact
- E = Open Store
- ESC = Return to Menu
